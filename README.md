Pikaday - With Time Picker
========

### Key Config Changes

```javascript
showTime: true
showMinutes: true
showSeconds: false
use24hour: false
incrementHourBy: 1
incrementMinuteBy: 1
incrementSecondBy: 1
autoClose: true
timeLabel: null // optional string added to left of time select
i18n: {
  en: {
    midnight: '', // Label for 0 AM
    noon: '' // Label for 12 AM
  }
}
```
